O projeto faz parte de uma das etapas do processo de nivelamento e capacitação da Liga Acadêmica de Sistemas Inteligentes (LASI)

....DESCRIÇÃO DO PROJETO...
Objetivo: Desenvolver um sistema que registra e armazena os nomes dos alunos de uma determinada turma bem como suas respectivas notas.
A linguagem de código ultiliZada no projeto foi PYTHON. para desenvolver o projeto ultilizamos o editor de código-fonte VISUAL STUDIO CODE (VS Code).

...DESCRIÇÃO DO FUNCIONAMENTO...
O código deve receber do usuário, as seguinguintes informações: 
*Número da turma;
*Número de Alunos;
*Nome de cada aluno;
*Notas de cada aluno listado.
E armazena-las em um arquivo.

Deverá retornar uma lista com os nomes, a maior e a menor nota e a média de cada aluno; também deverá retornar a média da turma.
